# Projeto TCA — Site Institucional para Lar de Idosos

Projeto desenvolvido como parte do **TCA (Trabalho de Conclusão de Ano)** no **1º ano do curso Técnico em Informática para Internet (ETEC)** em 2022.

A proposta consistia em analisar um site com oportunidades de melhoria e desenvolver uma versão mais organizada e funcional, com foco em **design, navegação e usabilidade**.

Nossa equipe escolheu como tema um **Lar de Idosos**, com a intenção de criar uma solução institucional clara e acessível, que pudesse futuramente ser disponibilizada à instituição.

---

## Principais melhorias implementadas

- Definição de uma identidade visual mais consistente, com paleta de cores harmoniosa  
- Estrutura de navegação mais objetiva, facilitando o acesso às informações  
- Organização do conteúdo em seções claras (história, serviços, eventos e contato)  
- Revisão textual para maior legibilidade e compreensão  
- Implementação de interações básicas com menus, botões e elementos clicáveis  
- Aplicação prática de fundamentos de desenvolvimento web aprendidos no início da formação
